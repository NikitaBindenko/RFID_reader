Инструкция для установки телеграм-бота:

1)Создать файл формата txt с названием users в директории со всеми файлами бота

2)в файле handlers.py в функции обработки команды watch, исправить в инициализации ком-порта название ком-порта, к которому в вашем случае подключён Arduino Uno

3)в файлах main.py и handlers.py исправить в переменной file открытие файла, для этого нужно указать корректный путь до файла users.txt
